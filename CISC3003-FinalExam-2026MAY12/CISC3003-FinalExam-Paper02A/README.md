# CISC3003 Final Exam Paper 02 - Scenario A

This project demonstrates:

- semantic HTML form design
- text inputs, textarea, select list, radio buttons, and checkboxes
- PHP processing and filter-based validation
- SQL injection protection with prepared statements
- MySQL schema creation and sample manual `INSERT INTO`

## Quick start

1. Copy this project folder into `C:\xampp\htdocs\`.
2. Create the database in phpMyAdmin by importing `database.sql`.
3. Optionally run `db/sample_insert.sql` to create a manual record for screenshots.
4. Open `http://localhost/CISC3003-FinalExam-Paper02A/`.
