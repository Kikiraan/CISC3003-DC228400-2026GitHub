# CISC3003 Final Exam Paper 02 - Scenario B

This project demonstrates:

- a contact form in HTML with client-side validation
- PHPMailer SMTP setup
- email sending and SMTP debug output
- post / redirect / get
- database logging of contact submissions

## Quick start

1. Copy this project to `C:\xampp\htdocs\`.
2. Import `database.sql` in phpMyAdmin.
3. Open `php/mail_config.php` and fill in your Gmail SMTP details.
4. Open `http://localhost/CISC3003-FinalExam-Paper02B/`.
