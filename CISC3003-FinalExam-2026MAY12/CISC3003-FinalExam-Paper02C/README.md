# CISC3003 Final Exam Paper 02 - Scenario C

This project demonstrates:

- secure registration and login
- browser validation and Ajax email checking
- email activation before login
- password reset by email
- a protected dashboard with user services

## Quick start

1. Copy this project to `C:\xampp\htdocs\`.
2. Import `database.sql` in phpMyAdmin.
3. Open `php/mail_config.php` and set the SMTP credentials plus the correct `base_url`.
4. Open `http://localhost/CISC3003-FinalExam-Paper02C/`.
